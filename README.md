# 记忆卡片系统

基于 Flask 的本地记忆卡片管理与每日背诵计划系统。

## 功能概览

- **记忆卡片库**
  - 创建、编辑、删除卡片（题目 / 关键点 / 详细内容）
  - 详情内容支持 **Markdown** 渲染
- **每日背诵计划**
  - 在 `记忆卡片` 页勾选卡片，设置开始/结束日期创建计划
  - 在 `今日计划` 页按乱序展示需要背诵的卡片
  - 学习时默认只看题目和关键点，点击“查看答案”显示 Markdown 详情
  - 将卡片标记为 **记住了 / 没记住**，计算今日进度
- **数据存储**
  - 使用 `data` 目录下的 JSON 文件本地存储：
    - `cards.json`：记忆卡片全集
    - `plans.json`：每日计划列表（含开始/结束日期和卡片 ID 列表）
    - `progress.json`：每天每张卡片的记忆状态

## 运行方式

1. 创建并激活虚拟环境（可选但推荐）

   ```bash
   cd memory-cards
   python -m venv .venv
   .venv\Scripts\activate  # Windows PowerShell
   ```

2. 安装依赖

   ```bash
   pip install -r requirements.txt
   ```

3. 启动服务

   ```bash
   python app.py
   ```

4. 浏览器访问

   在浏览器打开 `http://127.0.0.1:5000` 即可使用系统。

## 界面说明

- 页面顶部有 **“今日计划 / 记忆卡片”** 两个标签，切换两个模块。
- 整体采用渐变色现代化主题，支持手机和电脑的响应式布局。
